# Cycle 039: Evidence Consolidation

**Cycle:** 39
**Date:** 2026-03-15
**Type:** Evidence consolidation (ledger restructuring and final synthesis)

## Motivation

After 38 experimental cycles, the evidence ledger had grown to 625 lines (~42,800 tokens) and become extremely difficult to navigate. It contained:
- Dozens of struck-through/superseded entries inline with current findings
- 114+ open questions, the vast majority already answered with inline annotations
- Multiple revision histories embedded in the evidence table entries
- Experiment log summaries duplicating the individual experiment log files
- Inconsistent organization (findings added chronologically, not thematically)

The research program has addressed ALL core questions from research_spec.md:
- Cross-model replication: COMPLETE (4 families, 5 training variants)
- K-V functional dissociation: UNIVERSAL (5/5 models × 3 positions)
- Encoding taxonomy: COMPLETE (digital=Qwen-specific, analog=everyone else)
- Positional vs selectivity: RESOLVED (position dominates, selectivity adds zero)
- PGD null space: RESOLVED (K-specific, p=0.013, narrow)
- Literature convergence: CONFIRMED (5+ independent angles)

An evidence consolidation was the highest-value activity for this cycle.

## What Was Done

### Evidence Ledger Restructuring

The evidence ledger was completely reorganized from 625 lines of accumulated entries into a clean ~250-line document with the following structure:

1. **Summary** — Core hypothesis, verdict, and key numbers
2. **Established Findings** (7 sections) — Each finding with current status, key numbers, mechanistic explanation, and source experiments:
   - Text-Answer Dissociation (most replicated)
   - K > V Universal Hierarchy (routing > content)
   - PGD Null Space Is K-Specific
   - Encoding Taxonomy: Digital vs Analog
   - Positional Dissociation Dominates
   - Spatial Selectivity Is Weak
   - Literature Convergence
3. **Evidence Table** — Clean 21-row table with current status only (no strikethroughs or revision history)
4. **Disconfirmed Claims** — 12-entry table documenting what was wrong and why
5. **Open Questions** — Reduced from 114+ to 18 genuinely unanswered questions, organized by priority (high/medium/lower)
6. **Experiment Log** — Compact table format organized into 4 phases:
   - Phase 1: Establishing the Phenomenon (Cycles 1-9)
   - Phase 2: Positional Confound Resolution (Cycles 11-21)
   - Phase 3: K-V Functional Dissociation (Cycles 22-33)
   - Phase 4: Cross-Model Universality (Cycles 34-38)
7. **Narrative Synthesis** — "The story in five results" and implications for interpretability, safety, compression, and latent reasoning

### Key Organizational Decisions

1. **Removed all inline revision histories** — Previous entries had layers of "REVISED: ...", "FURTHER REVISED: ...", "~~strikethrough~~" annotations. The consolidated version shows only the CURRENT understanding.

2. **Separated answered from unanswered questions** — The 114+ open questions were triaged: ~96 already answered (annotations removed), 18 genuinely open questions retained and organized by priority.

3. **Created Disconfirmed Claims table** — Rather than leaving struck-through entries scattered throughout, collected all 12 major disconfirmed claims into a clean table with original evidence, disconfirming evidence, and resolution.

4. **Compact experiment log** — Replaced detailed per-experiment summaries (which duplicate experiment_log/exp_NNN.md files) with a phase-organized table showing experiment number, cycle, model, and one-line key result.

5. **Narrative synthesis** — Added a new "story in five results" section that presents the findings as a coherent argument rather than a chronological accumulation.

### What Was NOT Changed

- Individual experiment log files (experiment_log/exp_NNN.md) — these remain the authoritative record
- Literature notes (literature_notes/*.md) — unchanged
- Results directories (results/exp_NNN/) — unchanged
- research_spec.md, AGENT.md, config.env — never modified (per rules)

## Assessment

The consolidated ledger is approximately 4x shorter than the original while containing all essential information. A researcher or reviewer can now:
- Understand the current state of knowledge in the first 3 paragraphs
- Find any specific finding via the evidence table
- See what was wrong (and why) in the disconfirmed claims table
- Identify genuinely open questions (18 remaining)
- Trace any finding back to specific experiments via the experiment log table

The narrative synthesis section provides the coherent 5-point argument that connects the findings into a publishable story.
