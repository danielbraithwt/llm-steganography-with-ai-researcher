# Experiment 065: Early Answer Decodability (Linear Probe)

**Cycle:** 65
**Date:** 2026-03-21
**Type:** Phase 2 — Natural channel usage (observational evidence)
**Model:** Qwen/Qwen3-4B-Base
**Script:** `scripts/exp_065_early_decodability.py`

## Motivation

This is the first Phase 2 experiment. All 64 prior cycles used perturbation-based
methods (noise injection, PGD attacks, ablation, teacher-forcing corrupted text).
The critical gap: we have never shown that the hidden channel is used during
**normal, unperturbed generation**.

A skeptical reviewer can dismiss all Phase 1 evidence with: "You broke the KV
cache and things broke — obviously. The model is just reading its own text."

This experiment provides **observational** evidence by asking: does the KV cache
contain answer information **before** the text reveals it? If a linear probe on
K-cache activations can predict the final answer at intermediate CoT positions
where the text hasn't yet computed the answer, that's direct evidence the hidden
channel carries information beyond what's in the text.

This is Experiment A from the Phase 2 research spec — the highest priority
direction.

## Method

1. Generate 8-shot CoT on GSM8K (target: 80 correctly-solved problems) with
   Qwen3-4B-Base, collecting full KV cache via forward pass
2. Divide each CoT into 10 normalized position bins (10%, 20%, ..., 100%)
3. At each position bin, extract:
   - **K-cache features:** average K-vectors across positions in the bin,
     concatenated across all KV heads (dimension: n_kv_heads × head_dim)
   - **V-cache features:** same for V-vectors
   - **Text baseline:** cumulative mean of token embeddings from start to bin
     position (captures "what the text says so far" without attention routing)
4. Train ridge regression (RidgeCV with 5-fold cross-validation) to predict
   the final numeric answer (log-transformed) from features at each position
5. Measure Pearson correlation (r) between cross-validated predictions and
   actual answers
6. Probe at 4 layers (~25%, 50%, 75%, 100% of network depth)

**Controls:**
- **Shuffle control:** K features paired with shuffled answers → should give r ≈ 0
- **K vs V comparison:** expect K ≥ V (consistent with Phase 1 K>V hierarchy)
- **Layer sweep:** probe at 4 layers to find where answer info emerges

## Pre-Registered Predictions

### If hypothesis is TRUE (hidden channel carries answer info during normal generation):

- **K-probe reaches r ≥ 0.3 by 30-50% through CoT** (before the text has
  computed the final answer). The text baseline should not reach r ≥ 0.3 until
  70-90% through CoT (near the #### answer marker).
- **Early decodability gap ≥ 20% of CoT positions** (K-probe reaches threshold
  at least 2 position bins earlier than text baseline)
- **K-probe > V-probe** at most position bins (consistent with K>V from Phase 1)
- **Middle layers (50-75% depth) show the largest gap** — routing is established
  but the answer hasn't been committed to text yet
- **Shuffle control r ≈ 0** across all positions (validates the probe isn't
  overfitting to structural artifacts)

### If hypothesis is FALSE (model reads its own text, no hidden channel usage):

- **K-probe and text baseline track together** — both reach r ≥ 0.3 at
  approximately the same position. The KV cache learns the answer at the same
  rate as the text reveals it.
- **No early decodability gap** (gap < 10% of positions)
- **K-probe ≈ V-probe** (no routing vs content distinction)
- **Or: neither K-probe nor text baseline can predict the answer until very
  late in the CoT** (the computation genuinely unfolds step-by-step)

### This experiment is informative because:

1. **First observational (non-perturbation) test** of the hidden channel hypothesis
2. **Directly addresses the reviewer concern** — if K-probe decodes early, the
   channel carries answer info during normal generation
3. **Connects to Phase 1** — K>V hierarchy should manifest as K-probe > V-probe
4. **Clean methodology** — linear probe is a standard interpretability tool;
   cross-validated to prevent overfitting
5. **Multiple controls** (shuffle, K vs V, layer sweep) constrain interpretation
6. **Motivated by Boppana et al. (2026) "Reasoning Theater"** — models decide
   answers internally 80% of tokens before visible CoT
