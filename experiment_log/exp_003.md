# Experiment 003: Pruning Fraction Sweep — Diagnosing the Double Dissociation Failure

**Cycle:** 3
**Date:** 2026-03-14
**Type:** Experiment (diagnostic follow-up)
**Model:** Qwen/Qwen3-4B-Base
**Script:** `scripts/exp_003_pruning_sweep.py`

## Motivation

Experiment 002 found NO double dissociation: pruning 50% of answer-coupled positions yielded 100% accuracy, identical to pruning text-coupled or random positions. This challenges the spatial structure claim from Exp 5 (PGD perturbations correlate with answer-coupling at rho=0.78).

Two explanations:
- **(A) Methodological:** Zeroing KV entries is too weak. The model routes around zeros by attending to remaining positions. At 50% pruning, enough redundancy remains to preserve both channels.
- **(B) Genuine:** The spatial structure seen by PGD is an artifact of the optimizer (high-attention positions have more gradient signal), not genuine functional separation.

This experiment sweeps pruning fraction from 50% to 95% AND tests noise injection (replacing KV with Gaussian noise) as an alternative to zeroing. If (A), we'll find a cliff that differs between AC and TC; if (B), identical degradation profiles.

## Method

1. Load Qwen3-4B-Base, generate 8-shot CoT traces for 30 GSM8K problems
2. For each problem with correct baseline:
   a. Teacher-force trace, extract attention (last 4 layers)
   b. Classify positions as answer-coupled (AC) or text-coupled (TC)
   c. For each pruning fraction [0.5, 0.7, 0.8, 0.9, 0.95]:
      - Zero AC positions → measure accuracy + text loss
      - Zero TC positions → measure accuracy + text loss
      - Zero random positions → measure accuracy + text loss
   d. For noise injection at 0.9 pruning fraction:
      - Replace AC positions with Gaussian noise (matched norm) → measure
      - Replace TC positions with Gaussian noise → measure

### Conditions (per problem, per pruning fraction):
- Prune answer-coupled (zero)
- Prune text-coupled (zero)
- Prune random (zero)
- At 90%: Noise-inject answer-coupled
- At 90%: Noise-inject text-coupled

## Pre-Registered Predictions

### If hypothesis is TRUE (and exp_002 failure was methodological):
- **Pruning fraction cliff differs by position type:**
  - AC pruning: accuracy cliff at lower fraction (e.g., 80-90%)
  - TC pruning: accuracy cliff at higher fraction (e.g., 95%+) or not at all
  - Text loss cliff: TC pruning shows earlier/larger text loss increase than AC
- **Noise injection stronger than zeroing:** Noise at 90% AC positions should hurt accuracy more than zeroing at 90% AC positions (noise actively interferes rather than just removing)
- **Quantitative:** At 90% pruning, AC accuracy ≤ 70%, TC accuracy ≥ 85%

### If hypothesis is FALSE (genuine absence of spatial separation):
- AC and TC pruning show identical accuracy cliff profiles (same fraction, same magnitude)
- Noise injection at AC and TC positions produces equivalent effects
- No dissociation at any pruning fraction

### This experiment is informative because:
It directly addresses the most serious weakness in the current evidence: the exp_002 null result. By sweeping pruning fraction, we can determine whether the failure was due to insufficient ablation strength (methodological) or genuine absence of spatial structure (substantive).
