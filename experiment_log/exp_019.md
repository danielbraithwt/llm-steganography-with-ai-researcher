# Experiment 019: Position-Controlled Double Dissociation

**Cycle:** 19
**Date:** 2026-03-14
**Type:** Functional separability test — double dissociation
**Model:** Qwen/Qwen3-4B-Base
**Script:** `scripts/exp_019_double_dissociation.py`

## Motivation

After 18 cycles of investigation, the evidence picture has shifted dramatically:
- ALL selectivity-based destruction dissociations are positional confounds (exps 013, 016, 017)
- PGD perturbation-attention correlation dropped from 0.78 to 0.20 (exp 018)
- The null space EXISTS (Exp 4) but its spatial structure is much weaker than claimed

The **double dissociation experiment** has been the researcher's #1 priority since the initial directives, yet has never been properly executed. This experiment tests:
- Do AC-selective positions selectively carry ANSWER information?
- Do TC-selective positions selectively carry TEXT information?
- Is this separation genuine (survives position controls)?

Previous experiments measured only ONE dimension (accuracy). This experiment measures BOTH:
1. **Answer accuracy**: does noising these positions change the final answer?
2. **Text prediction accuracy**: does noising these positions degrade the model's ability to predict subsequent reasoning tokens?

A double dissociation would show: AC-noise hurts accuracy more than text, and TC-noise hurts text more than accuracy. This is the gold standard for functional separability.

## Method

1. Generate clean 8-shot CoT traces on Qwen3-4B-Base (GSM8K)
2. Compute AC scores (attention from last reasoning position) and TC scores (within-reasoning attention)
3. For each noise condition (strategy × noise fraction):
   - Select positions to noise
   - Step through reasoning tokens one at a time, noising selected positions causally
   - Measure text prediction accuracy (does the model correctly predict each next token?)
   - Generate answer from end of reasoning
4. Strategies:
   - **selac**: noise top AC-selective positions (unconstrained)
   - **seltc**: noise top TC-selective positions (unconstrained)
   - **random**: noise random positions
   - **q3_selac/q3_seltc**: within position quartile Q3 (0.50-0.75), noise by selectivity
   - **q4_selac/q4_seltc**: within position quartile Q4 (0.75-1.00), noise by selectivity
5. Noise type: replacement (norm-matched Gaussian)
6. Noise fractions: 5%, 10%

## Pre-Registered Predictions

### Scenario A: DOUBLE DISSOCIATION EXISTS (predicted 20% confidence)
Genuine functional separability — AC positions carry answer information, TC positions carry text information.

- **Unconstrained**: SelAC noise drops accuracy ≥15pp MORE than SelTC noise
- **Unconstrained**: SelTC noise drops text accuracy ≥5pp MORE than SelAC noise
- **Position-controlled (Q3/Q4)**: Within each quartile, AC-noise drops accuracy more AND TC-noise drops text more
- **Dissociation score**: SelAC positive (accuracy hurt more), SelTC negative (text hurt more)
- At 5% noise: SelAC accuracy ≤60%, SelTC accuracy ≥70%. SelAC text ≥90%, SelTC text ≤85%.

### Scenario B: NO DISSOCIATION — POSITIONAL ONLY (predicted 55% confidence)
All effects are driven by position. Within position bands, AC and TC noise have identical effects.

- **Unconstrained**: SelAC/SelTC accuracy difference EXISTS (same as exp_004/017)
- **Unconstrained**: SelAC/SelTC TEXT accuracy difference EXISTS (in the SAME direction — AC hurts text MORE)
- **Position-controlled (Q3/Q4)**: Within-quartile gaps < 5pp for BOTH accuracy AND text
- **Dissociation score**: similar sign for all strategies (no dissociation, just "hub" effects)
- AC positions are hubs: noising them hurts BOTH accuracy AND text equally

### Scenario C: PARTIAL DISSOCIATION — LATE POSITIONS ONLY (predicted 25% confidence)
Consistent with exp_017's finding: genuine selectivity effect exists in late half but not early half.

- **Q4**: AC-noise drops accuracy ≥10pp MORE than TC-noise (genuine within-quartile effect)
- **Q3**: AC vs TC noise within ≤5pp (no within-quartile effect)
- **Q4 dissociation**: opposite signs for AC vs TC dissociation score
- **Q3 dissociation**: same sign (no dissociation)

### This experiment is informative because:
1. Tests functional separability with BOTH dimensions (accuracy + text quality) — first time in this research program
2. Position-controlled from the start (Q3/Q4 strategies)
3. Double dissociation is the gold standard for functional separability in neuroscience
4. Doesn't depend on PGD convergence or correlational analysis
5. Directly addresses the researcher's top priority
6. If positive: strongest evidence yet for the hidden channel hypothesis
7. If negative: confirms that "answer-coupled" and "text-coupled" are not functionally distinct — the KV cache is a single computation channel, not two separable channels
